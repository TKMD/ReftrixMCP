## 日本語版

### 1. 商標について

「**Reftrix**」および「**ReftrixMCP**」（以下、総称して「本商標」）は、プロジェクト管理者（[@TKMD](https://github.com/TKMD)）の未登録商標（コモンロー商標）です。

本ソフトウェアはAGPL-3.0-onlyライセンスの下でオープンソースとして提供されていますが、オープンソースライセンスは本商標の使用を許諾するものではありません。ソフトウェアライセンスと商標権は独立した権利であり、本ガイドラインは商標の適切な使用を定めるものです。

本ガイドラインは、[Model Trademark Guidelines](https://modeltrademarkguidelines.org/)および[FOSSmarks](https://fossmarks.org/)の推奨事項を参考に作成されています。

### 2. 許可なく使用できる場合（Nominative Fair Use）

以下の使用は、事前の許可なく行うことができます。ただし、すべての場合において、商標権者との提携、後援、または公式な関係を示唆してはなりません。

#### 2.1 ソフトウェアへの言及

- 未改変のReftrixソフトウェアを正確に指し示すための使用
  - 例: 「このツールはReftrixMCPを使用しています」
  - 例: 「ReftrixのMCPツールと互換性があります」

#### 2.2 互換性・相互運用性の表示

- 自身のソフトウェアがReftrixと互換性があること、またはReftrixと連携して動作することを事実に基づき説明する使用
  - 例: 「Reftrix対応」「ReftrixMCP互換」
  - ただし、公式製品であると誤解される表現は禁止します

#### 2.3 学術・教育目的での使用

- 学術論文、技術ブログ、教育資料、プレゼンテーション等での引用・言及
  - 例: 論文中で「Reftrix [1] を用いてWebデザインのレイアウト分析を行った」と記載すること

#### 2.4 ニュース・論評での使用

- ニュース記事、レビュー、批評、比較記事等での事実に基づく言及

#### 2.5 未改変のソフトウェアの配布

- オープンソースライセンス（AGPL-3.0-only）に準拠し、ソフトウェアを改変せずに再配布する場合の商標使用

### 3. 許可が必要な使用

以下の使用には、事前にプロジェクト管理者の書面による許可が必要です。

- 本商標を含むドメイン名の登録（例: reftrix-cloud.com）
- 本商標を使用した商品・グッズの製作（Tシャツ、ステッカー等）
- カンファレンスやイベントの名称としての使用
- その他、上記「許可なく使用できる場合」に該当しない商業的使用

### 4. 禁止される使用

以下の使用は、いかなる場合も禁止されます。

#### 4.1 改変版への名称使用

- Reftrixのソースコードを改変した派生物に「Reftrix」または「ReftrixMCP」の名称を使用すること
  - フォークしたプロジェクトには、Reftrixとの区別が明確な別の名称を使用してください
  - ただし、「Based on Reftrix」「Fork of ReftrixMCP」のような出所の事実説明は許可されます

#### 4.2 公式であるかのような使用

- プロジェクト管理者の公式な承認・後援・提携があるかのように誤解させる使用
  - 例: 「Official Reftrix Partner」「Reftrix公認」等の表示（承認なしの場合）

#### 4.3 商用サービス名としての使用

- 「Reftrix」または「ReftrixMCP」をそのまま商用サービス・製品の名称として使用すること
  - 例: 「Reftrix Cloud」「ReftrixMCP Pro」等（許可なしの場合）

#### 4.4 混同を招く使用

- Reftrixの出所、品質、または性質について消費者・ユーザーに混同を生じさせるおそれのある使用
- 本商標の名称またはロゴを改変・変形して使用すること

### 5. 商標表示のガイドライン

本商標を使用する際は、以下のガイドラインに従ってください。

- 商標は常に形容詞として使用し、名詞や動詞として使用しないでください
  - ✅ 「Reftrix ソフトウェア」「ReftrixMCP ツール」
  - ❌ 「Reftrixする」「Reftrixs」
- 初出時には商標記号（™）を付記することを推奨します
  - 例: 「Reftrix™ は...」
- 商標を複数形、所有格、略語として使用しないでください

### 6. 法的根拠

本商標は、日本国における不正競争防止法（平成5年法律第47号）第2条第1項第1号および第2号に基づく保護、ならびに、各国のコモンロー（慣習法）上の商標権に基づいて保護されます。

This policy was developed with reference to the [Model Trademark Guidelines](https://modeltrademarkguidelines.org/) and [FOSSmarks](https://fossmarks.org/).

### 2. Uses That Do Not Require Permission (Nominative Fair Use)

The following uses are permitted without prior approval. In all cases, use must not suggest affiliation with, endorsement by, or sponsorship by the trademark owner.

#### 2.1 Referring to the Software

- Using the Marks to accurately identify the unmodified Reftrix software
  - Example: "This tool uses ReftrixMCP"
  - Example: "Compatible with Reftrix's MCP tools"

#### 2.2 Compatibility and Interoperability Statements

- Truthfully describing that your software is compatible with or interoperates with Reftrix
  - Example: "Reftrix-compatible" or "Works with ReftrixMCP"
  - However, use must not imply that it is an official product

#### 2.3 Academic and Educational Use

- Citation and reference in academic papers, technical blogs, educational materials, and presentations
  - Example: "We used Reftrix [1] to perform layout analysis of web designs"

#### 2.4 News and Commentary

- Factual references in news articles, reviews, critiques, and comparison articles

#### 2.5 Redistribution of Unmodified Software

- Use of the Marks when redistributing the software without modification, in compliance with the open source license (AGPL-3.0-only)

### 3. Uses That Require Permission

The following uses require prior written permission from the project maintainer:

- Registration of domain names containing the Marks (e.g., reftrix-cloud.com)
- Production of merchandise bearing the Marks (t-shirts, stickers, etc.)
- Use as the name of a conference or event
- Any other commercial use not covered under "Uses That Do Not Require Permission" above

### 4. Prohibited Uses

The following uses are prohibited under all circumstances.

#### 4.1 Use on Modified Versions

- Using "Reftrix" or "ReftrixMCP" as the name of derivative works that modify the Reftrix source code
  - Forked projects must use a distinct name that is clearly distinguishable from Reftrix
  - However, factual statements of origin such as "Based on Reftrix" or "Fork of ReftrixMCP" are permitted

#### 4.2 Implying Official Status

- Any use that suggests official endorsement, sponsorship, or affiliation with the project maintainer when none exists
  - Example: "Official Reftrix Partner" or "Reftrix Certified" (without authorization)

#### 4.3 Use as a Commercial Service Name

- Using "Reftrix" or "ReftrixMCP" as the name of a commercial service or product without permission
  - Example: "Reftrix Cloud" or "ReftrixMCP Pro" (without authorization)

#### 4.4 Confusing Use

- Any use likely to cause confusion regarding the source, quality, or nature of Reftrix
- Modifying or distorting the Marks or associated logos

### 5. Trademark Usage Guidelines

When using the Marks, please follow these guidelines:

- Always use the Marks as adjectives, not as nouns or verbs
  - ✅ "Reftrix software" or "ReftrixMCP tools"
  - ❌ "to Reftrix" or "Reftrixs"
- Include the trademark symbol (™) on first use where practical
  - Example: "Reftrix™ is..."
- Do not use the Marks in plural, possessive, or abbreviated forms

### 6. Legal Basis

The Marks are protected under Japan's Unfair Competition Prevention Act (Act No. 47 of 1993), Articles 2(1)(i) and 2(1)(ii), as well as common law trademark rights in applicable jurisdictions.
